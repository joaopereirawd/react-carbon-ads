# react-carbon-ads

add your [Carbon Ads](https://www.carbonads.net/) embed code with JSX

## Instalation

````
  npm i react-carbon-ads
  or 
  yarn add react-carbon-ads
  
````
## Usage

```js
import React from 'react';
import CarbonAds from "react-carbon-ads"

<CarbonAds 
    carbonUrl="CARBON_URL"
/>

```
## Props

- **Required props:**<br/>
`carbonUrl`

- **Optional props:**<br/>
`customClass`
